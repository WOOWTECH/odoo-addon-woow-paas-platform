# Task: Deployment verification

## Description

更新 Odoo 模組後，驗證 Odoo 模板在 Marketplace 中正確顯示，並實際透過平台部署一個 Odoo 實例以確認端對端流程正常運作。

## Acceptance Criteria

- [ ] 執行 `-u woow_paas_platform` 更新模組成功
- [ ] Marketplace 頁面顯示 Odoo 卡片（Web 分類下）
- [ ] 點擊 Odoo 卡片可見完整說明和配置表單
- [ ] 配置表單包含 email、password、demo data 欄位
- [ ] 成功部署 Odoo 實例（狀態變為 Running）
- [ ] 可透過 subdomain 訪問 Odoo Web UI
- [ ] 可使用配置的 email/密碼登入 Odoo

## Technical Details

### 驗證步驟

1. **模組更新**
   ```bash
   ./odoo-bin -c odoo.conf -u woow_paas_platform --stop-after-init
   ```

2. **Marketplace 驗證**
   - 訪問 `/woow`
   - 進入 Workspace → Services → Marketplace
   - 確認 Odoo 卡片顯示於 Web 分類
   - 確認卡片資訊正確（名稱、描述、tags）

3. **部署驗證**
   - 點擊 Odoo → 進入配置頁面
   - 填入 email 和密碼
   - 點擊 Launch
   - 等待部署完成（預計 3-8 分鐘）
   - 確認狀態變為 Running

4. **存取驗證**
   - 透過 subdomain 訪問 Odoo Web UI
   - 使用配置的 email/密碼登入
   - 確認基本 Odoo 功能可用

5. **清理**
   - 測試完成後刪除測試用服務
