# Task: Add Odoo template XML record

## Description

## Acceptance Criteria

- [ ] `cloud_app_odoo` record 已新增至 `cloud_app_templates.xml`
- [ ] Helm chart 使用 `oci://registry-1.docker.io/bitnamicharts/odoo`
