# 任務：更新 Header 元件

## 描述

增強 Header 元件以顯示更多使用者資訊：
1. 改進 Credits 徽章顯示（含加值按鈕）
2. 新增通知鈴鐺圖示（含未讀數量徽章）
3. 新增使用者名稱和方案類型
4. 新增使用者頭像（縮寫字母）

## 驗收條件

- [ ] Credits 徽章顯示餘額和錢幣圖示
- [ ] 通知鈴鐺圖示顯示，含紅點表示有未讀通知
- [ ] 顯示使用者名稱（如 "Sarah Wilson"）
- [ ] 顯示方案類型（如 "PRO PLAN"）
- [ ] 使用者頭像顯示姓名縮寫（如 "SW"）
