# 任務：更新 SCSS 樣式

## 描述

為新增的 Header 和 Dashboard 元素建立對應的 SCSS 樣式：
1. Header 使用者資訊區塊樣式
2. Credits 徽章樣式
3. 通知按鈕樣式
4. Dashboard 概覽卡片新樣式

## 驗收條件

- [ ] Credits 徽章樣式：圓角邊框、藍色主題、加號按鈕
- [ ] 通知鈴鐺：紅點徽章定位正確
- [ ] 使用者資訊：名稱粗體、方案標籤藍色小字
- [ ] 使用者頭像：圓形、縮寫字母置中
- [ ] Dashboard 卡片：支援 items 列表顯示、分數格式（如 5/10）
