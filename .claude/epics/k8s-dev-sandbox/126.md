# Task: Claude Code Skill 定義

## Description

建立 `.claude/skills/k8s-dev-sandbox.md` skill 定義檔，作為 `/k8s-sandbox` 命令的入口。Skill 解析用戶的子命令和參數，調用對應的 shell script。

## Acceptance Criteria

- [ ] `/k8s-sandbox create` → 呼叫 `k8s-sandbox-create.sh`
- [ ] `/k8s-sandbox destroy <name>` → 呼叫 `k8s-sandbox-destroy.sh`
- [ ] `/k8s-sandbox list` → 呼叫 `k8s-sandbox-list.sh`
- [ ] `/k8s-sandbox status [<name>]` → 呼叫 `k8s-sandbox-status.sh`
- [ ] `/k8s-sandbox logs <name>` → 呼叫 `k8s-sandbox-logs.sh`
- [ ] `/k8s-sandbox extend <name>` → 呼叫 `k8s-sandbox-extend.sh`
- [ ] `/k8s-sandbox test <name>` → 呼叫 `k8s-sandbox-test.sh`
- [ ] `/k8s-sandbox up` → 快捷建立（當前 branch, 預設 TTL）
- [ ] `/k8s-sandbox down` → 快捷銷毀（當前 branch 的沙盒）
- [ ] 無子命令時顯示 usage 說明

## Technical Details

Skill 檔案結構：
```markdown
# K8s Dev Sandbox

管理 Kubernetes 開發沙盒環境。

## Usage
/k8s-sandbox <command> [options]

## Commands
- create: 建立新沙盒
- destroy: 銷毀沙盒
- list: 列出所有沙盒
...

## Instructions
1. 解析用戶提供的子命令
2. 驗證必要參數
3. 呼叫對應的 shell script
4. 顯示執行結果
```

參考現有 `setup-end-to-end` skill 的撰寫模式。
