# Task: Responsive CSS fix and browser verification

## Description

修復 breadcrumb 的響應式排版問題，加入可點擊項目的 hover 樣式，並透過瀏覽器實際測試桌面版和手機版的 UI。

## Acceptance Criteria

- [ ] `.o_woow_breadcrumb_link` class 有 hover 樣式（cursor: pointer, color 變化）
- [ ] Header breadcrumb 在桌面版 (1280px+) 不溢出
- [ ] Page-level breadcrumb 在手機版 (375px) 不跑版
- [ ] 768px 斷點前後 UI 正確切換（sidebar ↔ bottom nav）
- [ ] 瀏覽器測試通過：所有主要路由 breadcrumb 可點擊且導航正確
- [ ] 手機版 breadcrumb 長文字正確截斷（ellipsis）

## Technical Details

### Files to modify
- `src/static/src/paas/styles/20_layout.scss`

### CSS 修復內容

#### 1. 可點擊 breadcrumb 樣式
```scss
.o_woow_breadcrumb_link {
    cursor: pointer;
    transition: color 0.2s;

    &:hover {
        color: $woow-primary;
    }
}
```

#### 2. Page-level breadcrumb responsive
```scss
.o_woow_breadcrumb {
    // 現有樣式保留...
    flex-wrap: nowrap;
    overflow: hidden;

    .o_woow_breadcrumb_current {
        overflow: hidden;
        text-overflow: ellipsis;
        white-space: nowrap;
        max-width: 200px;
    }
}

@media (max-width: 768px) {
    .o_woow_breadcrumb {
        .o_woow_breadcrumb_current {
            max-width: 150px;
        }
    }
}
```

#### 3. Back button touch target (mobile)
```scss
.o_woow_back_btn {
    min-width: 44px;
    min-height: 44px;
}
```
