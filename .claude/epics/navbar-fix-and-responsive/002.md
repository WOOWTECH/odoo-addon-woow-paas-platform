# Task: Page-level breadcrumb parent clickable navigation

## Description

在 6 個子頁面的 breadcrumb 中，將 parent 標籤（目前是純 `<span>`）改為可點擊元素，綁定 `router.navigate()` 到對應的父頁面。

## Acceptance Criteria

- [ ] WorkspaceDetailPage: "Workspaces" 可點擊 → `workspaces`
- [ ] WorkspaceTeamPage: workspace.name 可點擊 → `workspace/{id}`
- [ ] ServiceDetailPage: "Services" 可點擊 → `workspace/{id}`（回到 workspace detail）
- [ ] AppMarketplacePage: "Workspace" 可點擊 → `workspace/{id}`
- [ ] AppConfigurationPage: "Marketplace" 可點擊 → `workspace/{id}/services/marketplace`
- [ ] TaskDetailPage: "AI Tasks" 可點擊 → `ai-assistant/tasks`
- [ ] 所有可點擊 parent 有 hover cursor: pointer 視覺回饋

## Technical Details

### Files to modify (6 XML + 可能對應 JS)

#### 1. `pages/workspace/WorkspaceDetailPage.xml`
```xml
<!-- Before -->
<span class="o_woow_breadcrumb_parent">Workspaces</span>
<!-- After -->
<span class="o_woow_breadcrumb_parent o_woow_breadcrumb_link"
      t-on-click="() => this.props.router.navigate('workspaces')">Workspaces</span>
```
需確認 WorkspaceDetailPage 是否有 `router` prop，若沒有則需透過 `this.env` 或 import router。

#### 2. `pages/workspace/WorkspaceTeamPage.xml`
```xml
<span class="o_woow_breadcrumb_parent o_woow_breadcrumb_link"
      t-on-click="navigateToWorkspace" t-esc="workspace.name"/>
```
JS: `navigateToWorkspace() { router.navigate('workspace/' + this.props.workspaceId); }`

#### 3. `pages/service/ServiceDetailPage.xml`
```xml
<span class="o_woow_breadcrumb_parent o_woow_breadcrumb_link"
      t-on-click="navigateToWorkspace">Services</span>
```
JS: `navigateToWorkspace() { router.navigate('workspace/' + this.props.workspaceId); }`

#### 4. `pages/marketplace/AppMarketplacePage.xml`
```xml
<span class="o_woow_breadcrumb_parent o_woow_breadcrumb_link"
      t-on-click="navigateToWorkspace">Workspace</span>
```

#### 5. `pages/configure/AppConfigurationPage.xml`
```xml
<span class="o_woow_breadcrumb_parent o_woow_breadcrumb_link"
      t-on-click="navigateToMarketplace">Marketplace</span>
```
JS: `navigateToMarketplace() { router.navigate('workspace/' + this.props.workspaceId + '/services/marketplace'); }`

#### 6. `pages/task-detail/TaskDetailPage.xml`
```xml
<span class="o_woow_breadcrumb_parent o_woow_breadcrumb_link"
      t-on-click="navigateToTasks">AI Tasks</span>
```
JS: `navigateToTasks() { router.navigate('ai-assistant/tasks'); }`

### 注意事項
- 各子頁面可能直接 import router 或透過 props 取得
- 需逐一檢查每個頁面的 JS 確認 router 的取得方式
- 加上 `o_woow_breadcrumb_link` class 統一可點擊樣式
