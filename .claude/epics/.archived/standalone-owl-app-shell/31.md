### 常見問題修復

1. **Asset Bundle 未載入**
   - 檢查 `__manifest__.py` 配置
   - 確認 glob 路徑正確

2. **OWL 元件錯誤**
   - 檢查 template name 與 static.template 匹配
   - 確認 import 路徑正確

3. **樣式不生效**
   - 確認 SCSS 被 @import
   - 檢查 class 名稱拼寫

4. **Router 不工作**
   - 確認 hashchange 監聽器已註冊
   - 檢查 reactive 狀態更新
