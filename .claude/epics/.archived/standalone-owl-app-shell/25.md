# Task: AppShell Layout Components

## Description

建立 AppShell 佈局元件，包含 Sidebar（側邊欄）、Header（頂部導航）、和 Main Content 區域。這是應用的主要視覺框架。

## Acceptance Criteria

- [ ] 建立 `layout/app_shell/AppShell.js` 和 `.xml`
- [ ] 建立 `layout/sidebar/Sidebar.js` 和 `.xml`
- [ ] 建立 `layout/header/Header.js` 和 `.xml`
- [ ] 更新 Root 元件使用 AppShell
- [ ] Sidebar 顯示 Logo + 5 個導航項目 + 用戶資訊
- [ ] Header 顯示麵包屑 + Balance + 通知圖示 + 頭像
- [ ] Main Content 區域根據路由顯示對應內容
- [ ] 響應式：Mobile 時 Sidebar 隱藏

## Technical Details

### 檔案結構
```
static/src/paas/layout/
├── app_shell/
│   ├── AppShell.js
│   └── AppShell.xml
├── sidebar/
│   ├── Sidebar.js
│   └── Sidebar.xml
└── header/
    ├── Header.js
    └── Header.xml
```

### Sidebar 元件
- Logo: WoowTech 圖示 + 文字
- 導航選單：Dashboard, Workspaces, Deployments, Billing, Settings
- 當前選中項目高亮（primary 背景色）
- 底部：Help & Support 連結 + 用戶頭像/名稱

### Header 元件
- 左側：麵包屑（Home > 當前頁面）
- 右側：Balance 顯示（$5.00 Credits）+ 通知圖示 + 用戶頭像

### 佈局結構
```html
<div class="o_woow_app">
    <Sidebar />
    <main class="o_woow_main">
        <Header />
        <div class="o_woow_content">
            <!-- 頁面內容 -->
        </div>
    </main>
</div>
```

### 樣式要點
- Sidebar 寬度：260px
- Header 高度：80px
- 使用 flexbox 佈局
- Mobile 斷點：768px
