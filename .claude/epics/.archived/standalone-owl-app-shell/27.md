# Task: Base UI Components

## Description

建立基礎 UI 元件庫，包含 WoowIcon、WoowCard、WoowButton。這些元件將被 Dashboard 和 Workspace 頁面複用。

## Acceptance Criteria

- [ ] 建立 `components/icon/WoowIcon.js` 和 `.xml`
- [ ] 建立 `components/card/WoowCard.js` 和 `.xml`
- [ ] 建立 `components/button/WoowButton.js` 和 `.xml`
- [ ] 建立 `styles/components.scss` 元件樣式
- [ ] WoowIcon 封裝 Material Symbols，支援 size/filled props
- [ ] WoowCard 支援 slot 內容和 shadow 選項
- [ ] WoowButton 支援 variant (primary/secondary/ghost) 和 size props

## Technical Details

### 檔案結構
```
static/src/paas/components/
├── icon/
│   ├── WoowIcon.js
│   └── WoowIcon.xml
├── card/
│   ├── WoowCard.js
│   └── WoowCard.xml
└── button/
    ├── WoowButton.js
    └── WoowButton.xml
```

### WoowIcon 元件
```javascript
export class WoowIcon extends Component {
    static template = "woow_paas_platform.WoowIcon";
    static props = {
        name: String,
        size: { type: Number, optional: true },
        filled: { type: Boolean, optional: true },
        class: { type: String, optional: true },
    };
}
```
```xml
<span class="material-symbols-outlined"
      t-att-class="props.class"
      t-att-style="props.size ? 'font-size: ' + props.size + 'px' : ''"
      t-att-data-filled="props.filled ? '1' : '0'">
    <t t-esc="props.name"/>
</span>
```

### WoowCard 元件
```javascript
export class WoowCard extends Component {
    static template = "woow_paas_platform.WoowCard";
    static props = {
        shadow: { type: Boolean, optional: true },
        class: { type: String, optional: true },
        slots: { type: Object, optional: true },
    };
}
```
```xml
<div t-att-class="'o_woow_card ' + (props.shadow ? 'o_woow_card_shadow ' : '') + (props.class || '')">
    <t t-slot="default"/>
</div>
```

### WoowButton 元件
```javascript
export class WoowButton extends Component {
    static template = "woow_paas_platform.WoowButton";
    static props = {
        variant: { type: String, optional: true }, // primary, secondary, ghost
        size: { type: String, optional: true }, // sm, md, lg
        onClick: { type: Function, optional: true },
        slots: { type: Object, optional: true },
    };
}
```
```xml
<button t-att-class="'o_woow_btn o_woow_btn_' + (props.variant || 'primary') + ' o_woow_btn_' + (props.size || 'md')"
        t-on-click="props.onClick">
    <t t-slot="default"/>
</button>
```

### components.scss
```scss
// Icon
.material-symbols-outlined[data-filled="1"] {
    font-variation-settings: 'FILL' 1;
}

// Card
.o_woow_card {
    background: $woow-card-light;
    border-radius: $woow-radius-lg;
    border: 1px solid $woow-border-light;

    &.o_woow_card_shadow {
        box-shadow: $woow-shadow-card;
    }
}

// Button
.o_woow_btn {
    display: inline-flex;
    align-items: center;
    justify-content: center;
    gap: 0.5rem;
    font-weight: 600;
    border-radius: $woow-radius-default;
    transition: all 0.2s;
    cursor: pointer;
    border: none;

    &.o_woow_btn_primary {
        background: $woow-primary;
        color: white;
        &:hover { background: $woow-primary-hover; }
    }
    &.o_woow_btn_secondary {
        background: transparent;
        border: 1px solid $woow-border-light;
        color: $woow-text-secondary;
        &:hover { border-color: $woow-primary; color: $woow-primary; }
    }
    &.o_woow_btn_ghost {
        background: transparent;
        color: $woow-text-secondary;
        &:hover { background: rgba(0,0,0,0.05); }
    }

    &.o_woow_btn_sm { padding: 0.5rem 1rem; font-size: 0.875rem; }
    &.o_woow_btn_md { padding: 0.75rem 1.5rem; font-size: 0.875rem; }
    &.o_woow_btn_lg { padding: 1rem 2rem; font-size: 1rem; }
}
```
