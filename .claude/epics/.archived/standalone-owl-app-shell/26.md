# Task: SCSS Theme System

## Description

建立 SCSS 主題系統，從原型提取 Tailwind 主題變數（顏色、圓角、字體等），建立基礎樣式和元件樣式。此任務可與 Task 002, 003 並行開發。

## Acceptance Criteria

- [ ] 建立 `styles/variables.scss` 定義主題變數
- [ ] 建立 `styles/base.scss` 定義基礎樣式
- [ ] 建立 `styles/layout.scss` 定義佈局樣式
- [ ] 建立 `styles/main.scss` 作為入口 @import 所有
- [ ] QWeb 模板載入 Google Fonts（Manrope, Outfit）
- [ ] QWeb 模板載入 Material Symbols
- [ ] 所有樣式使用 `.o_woow_` 前綴避免衝突

## Technical Details

### 檔案結構
```
static/src/paas/styles/
├── variables.scss    # 主題變數
├── base.scss         # 基礎樣式（body, 滾動條等）
├── layout.scss       # 佈局樣式（app shell, sidebar, header）
└── main.scss         # @import 入口
```

### variables.scss
```scss
// 從原型提取的主題變數
$woow-primary: #5f81fc;
$woow-primary-hover: #4b6bfb;
$woow-background-light: #f5f6f8;
$woow-background-dark: #0f1323;
$woow-card-light: #ffffff;
$woow-card-dark: #1a1f36;
$woow-text-primary: #1A1C21;
$woow-text-secondary: #5f698c;
$woow-border-light: rgba(0, 0, 0, 0.06);
$woow-border-dark: rgba(255, 255, 255, 0.08);

// 圓角
$woow-radius-default: 0.75rem;
$woow-radius-lg: 1.25rem;
$woow-radius-xl: 1.5rem;
$woow-radius-2xl: 2rem;
$woow-radius-full: 9999px;

// 字體
$woow-font-display: 'Manrope', sans-serif;
$woow-font-body: 'Outfit', sans-serif;

// 陰影
$woow-shadow-soft: 0 4px 20px -2px rgba(0, 0, 0, 0.05);
$woow-shadow-card: 0 0 0 1px rgba(0,0,0,0.03), 0 2px 8px rgba(0,0,0,0.04);

// 間距
$woow-sidebar-width: 260px;
$woow-header-height: 80px;
```

### base.scss
```scss
.o_woow_app {
    font-family: $woow-font-display;
    background: $woow-background-light;
    color: $woow-text-primary;
    min-height: 100vh;
    -webkit-font-smoothing: antialiased;

    // 滾動條樣式
    ::-webkit-scrollbar {
        width: 6px;
        height: 6px;
    }
    ::-webkit-scrollbar-track {
        background: transparent;
    }
    ::-webkit-scrollbar-thumb {
        background: #cbd5e1;
        border-radius: 3px;
    }
}

// Material Symbols 設定
.material-symbols-outlined {
    font-variation-settings: 'FILL' 0, 'wght' 400, 'GRAD' 0, 'opsz' 24;
}
```

### QWeb 模板更新（字體載入）
```xml
<link href="https://fonts.googleapis.com/css2?family=Manrope:wght@200..800&amp;family=Outfit:wght@300..700&amp;display=swap" rel="stylesheet"/>
<link href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:wght,FILL@100..700,0..1&amp;display=swap" rel="stylesheet"/>
```
