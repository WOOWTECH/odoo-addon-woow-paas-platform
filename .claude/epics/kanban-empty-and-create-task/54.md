# Task: Integrate CreateTaskModal into ProjectKanbanPage

## Description

Add task creation capability to the ProjectKanbanPage by integrating the existing `CreateTaskModal` component. Add a "Create Task" button in the page header and improve the empty board state with better messaging.

## Acceptance Criteria

- [ ] "Create Task" button visible in kanban page header (next to project name)
- [ ] Clicking the button opens `CreateTaskModal` with `defaultProjectId` pre-filled
- [ ] After successful task creation, kanban board auto-refreshes with the new task
- [ ] Empty board state shows improved message instead of just "No stages configured"

## Technical Details

**Files to modify**:

1. **`src/static/src/paas/pages/project-kanban/ProjectKanbanPage.js`**:
   - Import `CreateTaskModal` from `../../components/modal/CreateTaskModal`
   - Add to `static components`: `{ WoowIcon, CreateTaskModal }`
   - Add state: `this.state = useState({ showCreateModal: false })`
   - Add methods:
     ```javascript
     openCreateModal() {
         this.state.showCreateModal = true;
     }
     closeCreateModal() {
         this.state.showCreateModal = false;
     }
     async onTaskCreated(task) {
         this.state.showCreateModal = false;
         await this._loadData();
     }
     ```

2. **`src/static/src/paas/pages/project-kanban/ProjectKanbanPage.xml`**:
   - Add button in `o_woow_kanban__page_header`:
     ```xml
     <button class="o_woow_kanban__create_btn" t-on-click="openCreateModal">
         <WoowIcon name="'add'" size="20"/>
         <span>Create Task</span>
     </button>
     ```
   - Add modal at bottom of template:
     ```xml
     <CreateTaskModal t-if="state.showCreateModal"
         defaultProjectId="props.projectId"
         onClose.bind="closeCreateModal"
         onCreated.bind="onTaskCreated"/>
     ```
   - Improve empty state text (line 97-101)
