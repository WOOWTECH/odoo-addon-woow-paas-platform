# Task: Support Tasks Page (Kanban)

## Description
建立 `SupportTasksPage` 頁面，以按專案分組的 Kanban 佈局展示支援任務列表。任務卡片顯示標題、優先級星號、截止日期、任務 ID、完成度、指派人頭像。支援「My & Shared Tasks」篩選、搜尋和多種視圖切換。

## Acceptance Criteria
- [ ] `SupportTasksPage.js` + `SupportTasksPage.xml` 建立
- [ ] Kanban 佈局（按專案分組的 columns）：
  - [ ] 每個專案為一個 column
  - [ ] Column header：專案名稱 + 任務數
- [ ] 任務卡片：
  - [ ] 任務標題
  - [ ] 優先級星號（1-3 星）
  - [ ] 截止日期
  - [ ] 任務 ID badge
  - [ ] 完成度 progress bar
  - [ ] 指派人頭像
- [ ] 「My & Shared Tasks」toggle 篩選
- [ ] 搜尋功能：按任務標題搜尋
- [ ] 點擊任務卡片導航到 `#/ai-assistant/tasks/:taskId`
- [ ] Empty state + Loading state
- [ ] `_support_tasks.scss` 建立

## Technical Details
- 使用 `support_service.js` 的 `fetchTasks(workspaceId)` 方法
- 任務資料來自 `project.task`（透過 `/api/support/tasks/<workspace_id>` API）
- Kanban columns 使用 CSS flexbox 水平排列
- 任務卡片可拖放（可選，v1 不需要）
- 「My Tasks」篩選：`assigned_to` 包含當前使用者
- 「Shared Tasks」篩選：workspace 內所有任務
- 參考設計圖中的 Kanban 頁面
