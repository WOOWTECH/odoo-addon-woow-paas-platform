# Task: Support Projects Page

## Description
建立 `SupportProjectsPage` 頁面，以 card grid 方式展示支援專案列表。每張專案卡片顯示名稱、狀態徽章、日期範圍、任務數、團隊成員頭像。支援搜尋、篩選、grid/list 視圖切換。提供「Apply for Project」申請加入功能。

## Acceptance Criteria
- [ ] `SupportProjectsPage.js` + `SupportProjectsPage.xml` 建立
- [ ] 專案卡片 grid 佈局：
  - [ ] 專案名稱
  - [ ] 狀態徽章（Active / Completed / On Hold）
  - [ ] 日期範圍（start - end）
  - [ ] 任務數量統計
  - [ ] 團隊成員頭像列（最多顯示 5 個 + "+N"）
- [ ] 搜尋功能：按專案名稱搜尋
- [ ] 篩選功能：按狀態篩選（All / Active / Completed）
- [ ] Grid / List 視圖切換
- [ ] 「Apply for Project」按鈕功能
- [ ] Empty state：無專案時顯示提示
- [ ] Loading state：載入中顯示 skeleton
- [ ] `_support_projects.scss` 建立

## Technical Details
- 使用 `support_service.js` 的 `fetchProjects(workspaceId)` 方法
- 專案資料來自 `project.project`（透過 `/api/support/projects/<workspace_id>` API）
- 卡片元件可重用 `WoowCard` 基礎，或建立專用 ProjectCard sub-component
- Grid 佈局使用 CSS Grid（responsive: 1-3 columns）
- 搜尋和篩選為前端 client-side filtering
- 參考設計圖 `resource/stitch_saas_web_app_ai_assistant_2026-02-09/` 中的專案頁面
