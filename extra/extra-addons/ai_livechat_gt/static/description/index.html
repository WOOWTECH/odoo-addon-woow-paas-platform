How It Works - 3 Simple Steps

1

Create AI Assistant

Go to Settings Technical AI Assistants. Create your AI assistant with ChatGPT, Claude, or Gemini configuration.

2

Configure Livechat

Enable AI assistant in your livechat channel. The AI will automatically engage visitors and answer questions 24/7.

3

Automatic Support

AI handles customer inquiries instantly. For complex issues, it seamlessly transfers to human agents with full context.

Technical Specifications

System Requirements

  • Odoo 18.0
  • PostgreSQL 13+ with pgvector extension
  • Livechat module enabled
  • Minimum 8GB RAM (16GB recommended)
  • Internet connection for AI APIs

AI Features & Tools

  • Semantic search across data sources
  • Vector embeddings with FastEmbed
  • Permission-aware data access
  • Multi-company security isolation
  • Intelligent conversation handling
