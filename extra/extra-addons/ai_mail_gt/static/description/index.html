How It Works - 3 Simple Steps

1

Create AI Assistant

Go to Settings Technical AI Assistants. Create your AI assistant with ChatGPT, Claude, or Gemini configuration.

2

Start Using

@mention AI in channels and use `$model/id` syntax to query specific records.

3

Get Intelligent Help

AI provides context-aware responses with record links. Get targeted information about specific records and data.

Technical Specifications

System Requirements

  • Odoo 18.0
  • PostgreSQL 13+ with pgvector extension
  • Discuss module enabled
  • Minimum 8GB RAM (16GB recommended)
  • Internet connection for AI APIs

AI Features & Tools

  • Record-specific queries with `$model/id` syntax
  • Data modification (create, update, delete records)
  • Method discovery and execution for custom business logic
  • Thread communication (read conversations and post messages)
  • Form integration with "Ask AI" button
  • @Mention detection and intelligent responses
  • Multi-AI support with specialized assistants
