[ADD] : 
Share Chat: Allow users to generate a shareable link for their chat. This typically involves creating a unique identifier for the chat session and providing a URL or export code that others can use to view the conversation.
Export Chat: Enable users to download their chat history in a file format (JSON). This preserves the entire conversation for offline access or archiving purposes.
Import Chat: Allow users to upload previously exported chat files to restore or continue a conversation. This typically involves parsing the file format and loading the chat history back into the application.
