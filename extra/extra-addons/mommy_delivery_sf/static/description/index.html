顺丰速运 / SF-EXPRESS

Powered by: 青岛欧姆网络科技(公众号:OdooHub)

本模块提供顺丰速运的承运商连接，用户可以方便地下单、跟踪物流和打印电子面单，支持港澳台及国际件。
This module provides SF-EXPRESS connector. Users can create orders, track route info, and print shipping labels. Supports Hong Kong, Macau, Taiwan, and international shipments.

视频演示请到B站搜索"顺丰速运"或公众号回复"顺丰"

功能简介/Features

配置顺丰速运参数

在这里配置客户号、校验码和月结单号。

发货单验证时自动生成运单跟踪号

运单号自动填入到跟踪单号字段内，并可跟踪物流信息。

也可在顺丰官网查询物流

验证后自动生成电子面单

电子面单可自定义。

跨境和多语言支持
