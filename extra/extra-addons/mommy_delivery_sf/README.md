# 顺丰物流对接

## 功能说明

1. 一键发货
2. 一键撤销发货单
3. 物流跟踪

## 配置说明

安装模块后，在系统设置中的配置丰桥API参数

* sf.clientcode: 丰桥分配的code
* sf.checkword： 丰桥分配的checkword
* sf.custid：月结客户的月结号

## 使用说明

###  发货

* 使用顺丰发货，必须要有源单据，即分拣单的源单据不能为空
* 发货单的客户地址必须完整，详细到街道，联系电话不能为空
* 发货仓库的地址必须完整，详细到街道，联系电话不能为空
* 发货成功后，运单号会自动填充到物流单号字段中。

### 物流信息查询

如果运单号是顺丰物流，模块会自动查询运单信息，并显示在发货单上。
